# Password Strength Checker

## Description

This project is a Python-based Password Strength Checker developed as part of my internship.

The program evaluates the strength of a password based on standard security rules and provides suggestions for improvement.

## Features

- Check minimum password length
- Detect uppercase letters
- Detect lowercase letters
- Detect numbers
- Detect special characters
- Display password strength
- Provide improvement suggestions

## Technologies Used

- Python 3
- Regular Expressions (re)
- getpass module

## Requirements

Python 3.x

## How to Run

python password_checker.py

## Example

Enter Password:
Hello@123

Password Strength:
VERY STRONG
